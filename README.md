## 📌 Key Insights

### Booking Channels

* Web channel generated the highest bookings.
* Web channel generated the highest revenue and profit.
* Travel Agent channel showed the highest cancellation rate.

### Room Categories

* Standard rooms contributed the largest booking volume.
* Deluxe rooms demonstrated the most stable demand.

### Hotel Ratings

* 4-Star hotels generated the highest revenue contribution.

### Seasonal Trends

* April was identified as the peak booking month.

### Revenue Leakage

* Cancelled bookings resulted in approximately **₹179.7 Million** in lost booking value.
* Estimated profit impact from cancellations was approximately **₹42.4 Million**.
